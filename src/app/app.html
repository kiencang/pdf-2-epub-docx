Cách chuyển PDF thành định dạng sách EPUB

1

Tải lên tài liệu PDF

Kéo thả trực tiếp tệp tin PDF hoặc nhấp chọn tệp từ máy tính. Ứng dụng sẽ tách các ảnh và văn bản từ tài liệu gốc để tái cấu trúc lại thành EPUB chuẩn.

2

Nhập API Key

Bản cấu trúc thô từ tài liệu PDF (văn bản chưa dàn trang, danh sách ảnh nằm lẻ) cần AI để ráp nối. Điền khóa API Key của bạn ở phần Nhập API Key để xử lý việc này.

3

Đổi định dạng sang EPUB

Hệ thống tự động gói dữ liệu thành tệp sách EPUB tiêu chuẩn, hỗ trợ tải về lưu trữ và xem tốt trên mọi máy đọc sách Kindle, Kobo, iPad. Và phục vụ mục đích dịch thuật tiện lợi có bảo toàn ảnh.
